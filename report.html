Methodology Note

Local Tailor uses a three-stage NLP pipeline running entirely on your local machine. Span extraction (deepset/roberta-base-squad2) identifies which part of each comment is relevant to each dimension. SetFit classification (few-shot fine-tuned all-MiniLM-L6-v2) assigns a dimension value from your defined categories using as few as 8 labeled examples per class. Sentiment analysis (cardiffnlp/twitter-roberta-base-sentiment) scores the extracted span, not the full comment, producing per-dimension sentiment rather than a single averaged score. No comment data is transmitted to any external server.
